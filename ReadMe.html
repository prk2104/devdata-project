Documentation: This is the documentation for the Shiny application that is part of the DSS Developing Data Products course. There aretwo selection boxes, a toggle box and a dropdown box in this application. The application produces boxplots based on the miles per gallon ad the predictor variable and three possible selected variables in the dropdown box. All you have to do is drop don the menu to select the regressor variable and toggle the switch to include outliers or not. Finally, the application show reactive output by changing the title as can be seen from the code in the github repository.
