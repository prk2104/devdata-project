Documentation: This is the documentation for the Shiny application that is part of the DSS Developing Data Products course. There are three selection boxes, in this application. The application produces scatter plots based on the chosen regressor variables and out come variable. All you have to do is drop down the menu to select the regressor variable and outcome variable and select the color of the points fromt he dropdown box. Finally, the application show reactive output by changing the title as can be seen from the code in the github repository.
